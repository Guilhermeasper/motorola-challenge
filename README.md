# Motorola Challenge

## Atividade Complementar 3ª Etapa – Processo Seletivo Edital 32/2020

Programa orientado a objetos em python que simula o comportamento de transição de telas de um celular em uma ligação utilizando design patern state.

## Executando

Clone o repositório e dentro da pasta raiz do repositório, execute:

    python main.py -input="caminho/para/o/arquivo/inputs.txt" -output="caminho/para/o/arquivo/outputs.txt"
    
## Formato da entrada

As seguintes entradas são consideradas amigaveis para o programa:

- BL (Botão de ligar)
- BD (Botão de desligar)
- AD (Avatar do destinatário)
- PQR (Pop-up de queda de rede)
- PNA (Pop-up de não atendido)
- PFC (Pop-up de fim de chamada)

Demais entradas no programa retornam "Invalid input"
